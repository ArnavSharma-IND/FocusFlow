# 🌊 FocusFlow — Modern Animated Todo & Productivity App

> **Focus better. Organize smarter. Achieve more.**

**FocusFlow** is a premium productivity application designed to help users manage tasks efficiently with a beautiful UI, smooth animations, and powerful productivity tools. Built with modern web technologies, it delivers a seamless experience across desktop and mobile devices.

---

## ✨ Features

### 📝 Smart Task Management

* Create, edit, and delete tasks
* Mark tasks as completed
* Undo deleted tasks
* Drag & drop task reordering
* Bulk actions for multiple tasks

### 📅 Rich Task Details

* Task title & description
* Due dates
* Priority levels:

  * 🟢 Low
  * 🟠 Medium
  * 🔴 High
* Categories & tags
* Completion tracking

### 🔍 Search & Filtering

* Real-time search
* Filter by:

  * Pending tasks
  * Completed tasks
  * Priority
  * Category
  * Due date
* Sort tasks by:

  * Newest
  * Oldest
  * Deadline
  * Priority

### 📊 Productivity Dashboard

* Total tasks
* Completed tasks
* Pending tasks
* Productivity score
* Circular progress indicator
* Weekly productivity insights

### 🎨 Premium UI/UX

* Modern glassmorphism design
* Dark & light themes
* Fully responsive layout
* Floating Action Button (FAB)
* Minimal and elegant interface

### ⚡ Smooth Animations

* Animated task creation
* Swipe-to-delete effects
* Hover interactions
* Drag-and-drop physics
* Page transitions
* Completion celebrations 🎉

### 💾 Local Storage

* Automatic task persistence
* Offline support

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/focusflow.git
cd focusflow
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

Open your browser and visit:

```bash
http://localhost:5173
```
